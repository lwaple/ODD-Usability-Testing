Patent grant optical character recognition

Patent grant optical character recognition (OCR) text (1920 - 1979) (OCR text)

Contains patent grant text produced by an Optical Character Recognition (OCR) process using the U.S. Patent Grant Image data from January 1920 through December 1979 as input. The file format is modified Patent Full-Text/APS (a.k.a., modified Green Book), ASCII text.
